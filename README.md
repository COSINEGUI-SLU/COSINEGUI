# COSINE App

COSINE is a standalone Java-based graphical application for comparing experimental and predicted MS/MS spectra using cosine similarity scoring. It supports tolerance-based peak matching, mirror plot visualization, and CSV-based spectral input.

## Download

Download the latest version from the **Releases** section.

## Requirements

- Java 17 or later

## How to Run

After downloading `COSINEGUI app`, install in windows.
